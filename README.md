# Train-System
T# 🚆 Train Schedule and Reservation Management System

This project is a Train Schedule and Reservation Management System developed for the Software Engineering course. It allows administrators to manage train schedules, handle reservations, and validate seat availability to prevent overbooking. The system was implemented using Java and Java Swing.

## Features

* User login system
* Add and view train schedules
* Reservation management
* Seat availability validation

## How to Run

1. Open the project in IntelliJ or Eclipse
2. Run MainSystem.java
3. Login using:
